Alpha Leak is a production-grade, multi-phase trading intelligence system built for Pump.fun on Solana. It processes every trade, token creation, and graduation event in real time, runs them through layered analysis — wallet scoring, adversarial detection, ML inference, and market regime classification — and executes trades atomically when high-conviction signals emerge.

## How the system works

Alpha Leak runs as a single process with 30+ background services coordinated through a database and cache layer. Events flow through four named phases, each building on the data produced by the previous one.

<Steps>
  <Step title="Ingest on-chain events">
    The `EventProcessor` receives every trade, token creation, and graduation event the moment it lands on-chain — no polling, no batching. Events are decoded and fanned out simultaneously to all relevant services.
  </Step>
  <Step title="Score wallets and build intelligence">
    Phase 1 computes a 7-component alpha score for every wallet. Phase 2 classifies token lifecycle state, detects coordinated bundles, scores creator risk, and builds a wallet co-occurrence graph.
  </Step>
  <Step title="Run ML inference">
    Phase 3 scores signals every 5 seconds using ONNX LightGBM models with Platt-calibrated probabilities. Adversarial detection, copy-trade classification, and market regime run concurrently.
  </Step>
  <Step title="Execute trades atomically">
    The live trader subscribes to the signal channel, selects a strategy based on phase and ML score, and executes buys and sells on the Pump.fun bonding curve as atomic transaction bundles.
  </Step>
</Steps>
